## Installation

The program can be compiled by g++ compiler in Linux system. Before the compilation, you must install the SFML librariy using the following instruction in terminal : 

```sh
$ sudo apt-get install libsfml-dev
```

## Compilation

```sh
$ make
$ ./build/tetris
```
